# Samples

- .NET 6
- node.js v16.14.2
- yarn v1.22.17

## Server
```
$ cd samples/SampleServer
$ dotnet run --project SampleServer.csproj
```


## TypeScript Console App



```
$ cd samples/console.typescript
$ yarn install
$ yarn start
```

## C# Console App

```
$ cd samples/Console.CSharp
$ dotnet run --project Console.CSharp.csproj
```
